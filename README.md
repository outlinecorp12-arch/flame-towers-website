# Flame Towers Baku - Luxury Mixed-Use Development Website

## 🏢 Project Overview

A comprehensive, modern website for **Flame Towers**, Baku's most iconic mixed-use development featuring luxury residences, premium offices, and a world-class shopping mall.

### Key Information
- **Location**: Baku Boulevard, Baku, Azerbaijan
- **Height**: 182 meters (3 iconic towers)
- **Completed**: 2012
- **Designed by**: HOK International
- **Architecture**: Inspired by flames ("Land of Fire" reference)
- **Features**: Blue reflective glass facade, LED lighting system
